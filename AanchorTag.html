Anchor tag

Anchor Tag- is used to link the pages. or mention the link of other page on your website.
Anchor tag has attributes as
Anchor Tag Attributes with values

  • target = "blank" => linked page open on another page
  • target = "self" => linked page open on same page
  • target = "parent "=> linked page open on parent page
  • target = "top => linked page open on parent pages top "
